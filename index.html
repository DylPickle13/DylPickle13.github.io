EnGem Overview

EnGem is a local assistant and automation framework that connects a streaming LLM (Gemini) to host-side tooling and exposes a Discord bot interface. It provides:

  • LLM orchestration and sub-agent execution plans
  • A Discord bot for text chat, scheduling, and execution-plan progress tracking
  • Persistent semantic memory backed by ChromaDB
  • Modular skills for browser automation, running code, notebooks, and media generation
  • Output capture and artifact relocation into generated_files/

This documentation outlines how to get started, where important pieces live, and common development tasks directly derived from the source README.md.

Quick start

1 Create environment & install dependencies:

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

2 Install Playwright browser binaries:

python -m playwright install

3 Set required environment variables:

  • DISCORD_BOT_TOKEN — Discord bot token (required to run the bot).
  • PAID_GEMINI_API_KEY — Gemini API key used by the LLM clients/tools.
  • DISCORD_ALLOWED_CHANNELS — Comma-separated allowed channel names.
  • Optional: REPO_PATH, CRON_JOB_HOUR, MODEL — see config.py.

4 Run the Discord bot:

python discord_bot.py

This starts the bot. The bot writes per-channel conversation history files to memory/channel_history/ and uses llm.py to orchestrate intent classification and sub-agent execution.

Data & Storage

  • Conversation histories: memory/channel_history/*.md (one file per Discord channel).
  • Vector DB: memory/vector_db/ (ChromaDB). Generates embeddings via configured Gemini service. Semantic memories in engem_memory_semantic_gemini2.
  • Generated outputs: generated_files/.
  • Sub-agent execution orders: sub-agents/ (JSON files).
  • Agent instructions: agent_instructions/ (templates).

Troubleshooting

Playwright Issues

If Playwright actions fail, ensure browsers are installed:
python -m playwright install

API Errors

If Gemini/Google APIs error frequently, api_backoff.call_with_exponential_backoff retries on transient errors; verify your API keys in the environment.

Bot Connection

If the bot does not receive messages, meticulously verify DISCORD_BOT_TOKEN and DISCORD_ALLOWED_CHANNELS are correctly set.

Developer notes

  • Manager/Planner: enforces a final `Reviewer` agent that must print <yes> when the task is complete.
  • Tool discovery: LLM sub-agents automatically call helpers in tools/. Ensure valid function signatures.
  • Artifact relocation: tools/run_python.py snapshots the repo root, runs code, and moves generated artifacts into generated_files/.
  • ChromaDB: Embeddings generated via Gemini. Make sure a paid API key is available. Back up memory/vector_db/.
