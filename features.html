Engine Capabilities

A modular ecosystem designed for complex tasks.

use_browser

Seamless web navigation, complex UI interaction, and deep data extraction using advanced LLM reasoning to parse DOM structures efficiently.

run_python

Sandboxed Python execution for robust logic handling, data manipulation, and automated workflow scripts.

generate_image

High-fidelity image synthesis using Imagen 3 for creative generation.

generate_video

Video rendering via Google Veo, translating prompts to cinematic clips.

run_notebook

Jupyter Notebook automation via papermill. Execute complex data science pipelines seamlessly and capture visual outputs.

run_google_search

Direct integration with Google Search APIs for real-time information retrieval.

access_google_drive

Full suite of Drive functions: upload, download, search, and manage files securely.

collect_generated_media

Centralized media repository management. Tracks and organizes generated assets cleanly.
