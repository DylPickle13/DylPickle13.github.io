Agentic Capabilities

Modular skills enabling intelligent autonomous execution.

🌐

use_browser

Full autonomous web navigation. The agent can seamlessly browse pages, extract context, interact with UI elements, and summarize complex web architectures in real-time.

🐍

run_python

Secure, local Python code execution. Enables the engine to dynamically process data, construct pipelines, and manipulate OS-level resources on the fly.

🎨

generate_image

Direct integration with cutting-edge visual synthesis models. Produce high-fidelity images directly from conversational prompts.

🎬

generate_video

Transform text into motion. Generate short, stunning video clips from descriptive prompts to enhance multi-modal outputs.

🔍

run_google_search

Real-time information retrieval. Bypasses static knowledge limits by aggregating and summarizing live data directly from Google Search.

📓

run_notebook

Data Science Sandbox. Dynamically execute entire Jupyter notebooks (using Papermill) for complex, multi-stage analytical workflows and persistent data visualization.

📁

collect_generated_media

Automated media indexing. Instantly retrieve, sort, and manage all generated visual and video assets across the workspace.
