Engine Capabilities

A modular ecosystem designed for complex, agentic tasks.

use_browser

Seamless web navigation, complex UI interaction, and deep data extraction using advanced LLM reasoning to parse DOM structures efficiently.

run_python

Sandboxed Python execution for robust logic handling, data manipulation, and automated workflow scripts. Run operations safely.

generate_image

High-fidelity image synthesis for creative generation and visual content creation directly from prompts.

generate_video

Cinematic video rendering translating complex scene descriptions into high-quality motion clips.

run_notebook

Jupyter Notebook automation. Execute complex data science pipelines seamlessly, track metrics, and capture visual outputs autonomously.

run_google_search

Direct integration with Google Search APIs for real-time information retrieval and summarization.

access_google_drive

Full suite of Google Drive functions: upload, download, search, and manage files securely via API.

collect_generated_media

Centralized media repository management. Automatically tracks, organizes, and retrieves generated assets cleanly for downstream use.
