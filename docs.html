Introduction

EnGem is a local assistant and automation framework that connects a streaming LLM to host-side tooling and exposes a Discord bot interface.

It provides sub-agent execution plans, lightweight skills, persistent conversation memory, a Chroma vector DB, and a Discord bot for chat, voice, and tool orchestration.

Quick Start

Create and activate a virtual environment, then install dependencies:

python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

Run the Discord bot:

python discord_bot.py

Configuration

Configure required environment variables (examples):

  • DISCORD_BOT_TOKEN: Discord bot token (see config.py)
  • GEMINI_API_KEY: API key for Gemini / LLM provider
  • Optional: DISCORD_ALLOWED_CHANNELS, VOICE_TOOL_TARGET_CHANNEL_NAME, MODEL

Skills & Tools

EnGem executes tools located in the skills/ folder dynamically. Common tools include use_browser, run_python, and run_notebook.

Architecture

The system utilizes an orchestrator model and sub-agent architecture to plan complex tasks, execute them in parallel or serial configurations, and verify outputs.
