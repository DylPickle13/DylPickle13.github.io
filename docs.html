Documentation

Setup, architecture, and extension guides.

Quick Start

Initialize the environment and launch the agent framework:

python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

Core Architecture

EnGem relies on modular components that interact seamlessly:

  • LLM Layer: Google Gemini handles multi-modal reasoning.
  • Tool/Skill System: Sandboxed execution for Python, browser control, and API interactions.
  • Vector Memory: ChromaDB maintains persistent conversational context.
  • Orchestrator: Manages parallel/serial agent execution plans dynamically.

Adding Skills

Create a python module in `/skills` exposing a function. EnGem will auto-detect and register it as an executable tool.
